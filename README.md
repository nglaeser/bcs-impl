# Blind Conditional Signatures (BCS)

## Dependencies
- `emp-ag2pc`: Follow the instructions [here](https://github.com/emp-toolkit/emp-ag2pc).
- Run `pip install -r requirements.txt`

## Install
```
mkdir build
cd build
cmake ..
cd ..
make -C build main
```

## Usage
```
./build/bin/main [arguments]
```
